# Team-5-achievers
CSC 131 Client project

The CPR Lifeline Automation System automates student registration processing for CPR training courses such as
BLS, ACLS, and PALS. The project integrates Azure Outlook, AHA Atlas automation, Google Sheets synchronization,
SFTP file uploads, and automated reminder systems into a single platform.
The system extracts student registration information from emails, processes enrollment data, generates
RQI-compatible CSV files, uploads records to SFTP servers, and provides analytics using a PyQt5 dashboard
interface.
